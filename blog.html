My Projects

A collection of software development and academic projects.

BookCorner

A web-based platform developed using CodeIgniter 4, PHP, MySQL, HTML, CSS, and JavaScript that facilitates the buying and selling of used textbooks and academic materials among university students, promoting affordable access to educational resources.

EatSmart

A web-based calorie tracking system developed using CodeIgniter 4, PHP, MySQL, HTML, CSS, and JavaScript that enables users to monitor their daily calorie intake, track nutritional information, and support healthier dietary habits.

University Event Management

A web-based event management system developed using PHP, MySQL, HTML, CSS, and JavaScript that facilitates the planning, organization, and management of university events, ensuring efficient coordination between organizers and participants.
